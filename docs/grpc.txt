gRPC Documentation
==================

Location data definition:

message LocationMessage {
    int32 person_id = 1;
    string latitude = 2;
    string longitude = 3;
}

Example:

{
    person_id=1,
    latitude=48.113704,
    longitude=11.637020
}



The gRPC server is hosted on the location-ingestor microservice on port 5005. 
One can use grpc_server.py in the microservice to illustrate sending location data to the server.

To test, run these:


export LOCATION_INGESTER_POD=$(kubectl get pods --namespace default -l "app=udaconnect-location-ingester" -o jsonpath="{.items[0].metadata.name}")

kubectl exec -it $LOCATION_INGESTER_POD sh



The above command is to enter the location-ingestor pod. 
Once inside the pod, you can run the following command:


python writer.py
